Objective/ Problem Statement

The objective of this project was to create an advanced task and diary management application that enhances personal productivity and encourages regular journaling. The challenge was to design a solution that integrates seamlessly with popular tools, ensures data security, and provides a user-friendly experience across multiple devices.

What I did

We developed a comprehensive application combining task management and diary features. The application allows users to create and categorize tasks, track their progress, and archive completed tasks. Additionally, it includes a diary section for daily journaling, with secure access and integration with Google Sheets/Docs for data backup and accessibility. The project was built using front-end technologies like HTML, CSS, and JavaScript, while Python was employed for back-end functionality.

Tools

  • Front-End: HTML, CSS. Bootstrap, Javascript
  • Data Integration: Google Sheets/Docs, Microsoft ToDo
  • Security: Data encryption for sensitive information
  • Responsive Design: Ensuring accessibility across devices

    • Key Features

  • Task Creation and Categorization: Users can create tasks with specific deadlines, priorities, and categories (e.g., Work, Personal)
  • task Tracking: Track the status of tasks, with options to mark them as completed or in progress, and archive completed tasks for later review.
  • Diary Integration: Allows for daily journal entries with secure, password-protected access
  • Responsive Design: Accessible on desktops, tablets, and smartphones
  • Data Integration: Syncs with Google Sheets/Docs and Microsoft ToDo for seamless data management and backup.
  • Enhanced User Experience: Intuitive interface with interactive elements such as drag-and-drop task reordering

  • Diary Management: Write, store, and search diary entries with secure access

  • Interactive UI: JavaScript-driven features enhance user interaction and experience.
