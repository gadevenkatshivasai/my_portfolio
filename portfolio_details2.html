CYBER THREAT DETECTION

Objective/Problem Statement

The primary challenge in cybersecurity is developing an automated and effective technique for detecting cyber threats. Existing methods often struggle to differentiate between true positive and false positive alerts, leading to inefficiencies and increased response times for security analysts. The objective of this project was to create a robust AI-based system capable of accurately detecting cyber threats using advanced neural network techniques.

What We Did

We developed an AI-SIEM (Security Information and Event Management) system that leverages artificial neural networks (ANNs) for enhanced cyber-threat detection. The system converts a multitude of collected security events into individual event profiles and employs deep learning methods to identify potential threats more effectively. The system focuses on improving the accuracy of alerts by distinguishing between true positives and false positives.

Tools Used

  • Data Processing & Profiling: Python, Pandas
  • Deep Learning Frameworks: TensorFlow, Keras
  • Machine Learning Models: Support Vector Machine (SVM), k-Nearest Neighbors (k-NN), Random Forest (RF), Naive Bayes (NB), Decision Tree (DT)
  • Datasets: NSL-KDD, CICIDS2017, and two real-world datasets
  • AI- Model ully Connected Neural Networks (FCNN), Convolutional Neural Networks (CNN), Long Short-Term Memory networks (LSTM)

    • Key Features

  • Event Profiling: Converts raw security events into structured event profiles, enabling more efficient data preprocessing and analysis.
  • Multiple Neural Networks: Utilizes FCNN, CNN, and LSTM to analyze event profiles from different perspectives and improve detection accuracy.
  • Alert Accuracy: Focuses on reducing false positives to ensure that alerts generated are actionable and accurate.
  • Benchmarking Evaluated against traditional machine learning methods to demonstrate the superiority of the proposed approach.

    • Functionality

  • Data Ingestion: The system ingests large volumes of security event data and preprocesses it into structured event profiles.
  • Threat Detection: Employs deep learning models to analyze the event profiles and detect potential threats.
  • Alert Generation: Generates alerts based on the analysis, with a focus on minimizing false positives.
  • Performance Comparison: Compares the effectiveness of neural network-based detection with traditional machine learning models.

    • Project Implementation

  • Data Collection & Preprocessing: Collected and preprocessed data from the NSL-KDD and CICIDS2017 datasets, along with real-world data, to create a comprehensive dataset for training and evaluation.
  • Model Development: Developed and trained multiple neural network models (FCNN, CNN, LSTM) using the preprocessed event profiles.
  • Evaluation:Conducted experiments comparing the AI-based system to traditional machine learning methods (SVM, k-NN, RF, NB, DT).
  • System Integration: Integrated the trained models into the AI-SIEM system to automate the cyber-threat detection process.

    • Challenges & Solutions

  • Challenge Differentiating between true positive and false positive alerts, which is crucial for reducing unnecessary load on security analysts
  • Solution Implemented event profiling techniques combined with deep learning models to enhance the accuracy of alert classification.
  • ChallengeEnsuring the system performs well across different datasets, including real-world data.
  • Solution Trained and validated the models on both benchmark datasets and real-world data to ensure robustness and generalizability

    • Conclusion

    The AI-SIEM system developed in this project successfully addresses the challenge of automated and effective cyber-threat detection. By employing advanced neural network techniques and focusing on accurate alert generation, the system outperforms traditional machine learning methods in both benchmark and real-world scenarios. This system not only enhances the efficiency of cybersecurity operations but also aids in the rapid and accurate response to potential threats, proving its value as a learning-based model for network intrusion detection.
